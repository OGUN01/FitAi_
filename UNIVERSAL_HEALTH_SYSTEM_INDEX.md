## KEY FEATURES SUMMARY

### 1. Auto-Detection Framework
- **Ethnicity:** Detects from country/region (85-90% confidence)
- **Climate:** 4 zones (tropical, temperate, cold, arid)
- **Only asks user when confidence < 70%**

### 2. Universal BMI Classification
- **7 populations:** Asian, Caucasian, Black/African, Hispanic, Middle Eastern, Pacific Islander, Mixed
- **WHO-approved cutoffs** for each population
- **Example:** Asian BMI 23 = Overweight (vs 25 in general WHO)

### 3. Multi-Formula BMR System
- **4 formulas:** Mifflin-St Jeor, Katch-McArdle, Cunningham, Harris-Benedict
- **Auto-selection** based on data availability
- **Accuracy:** ±5% (vs ±10% single formula)

### 4. Climate-Adaptive Calculations
- **TDEE:** +5% tropical, +15% cold, baseline temperate
- **Water:** +50-70% tropical/arid, -10% cold
- **Regional fine-tuning** (e.g., Indian states)

### 5. Diet-Type Protein Scaling
- **Vegetarian:** +15% protein (incomplete proteins)
- **Vegan:** +25% protein (lower bioavailability)
- **Research-backed adjustments**

### 6. Heart Rate Zone Formulas
- **3 methods:** Karvonen (personalized), Tanaka (older/fit), Gulati (female-specific)
- **Auto-selection** based on age/gender/fitness

### 7. Experience-Based Muscle Gain
- **Beginner:** 0.75-1.25 kg/month (male)
- **Intermediate:** 0.4-0.6 kg/month
- **Advanced:** 0.2-0.25 kg/month
- **Elite:** 0.1 kg/month

### 8. Flexible Validation
- **No hard blocks** - tiered warnings only
- **⚠️ Caution** → **⚠️⚠️ Warning** → **⚠️⚠️⚠️ Severe**
- **User empowerment with education**
